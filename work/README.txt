Ограничение на имя входного файла вместе с <.> и расширением 32 символа
Ограничение на расширение входного файла 8 символов.
Временный фаил для записи псевдобинарного кода <temp.101>. После записи выходного файла <temp.101> не удаляется.
Выходной сжатый фаил имеет расширение hfm.
После декомпрессии возвращается прежнее расширение файла. При совпадении имён затирает одноимённый фаил.
